# 团队logo
![这是我们的团队Logo](./team24logo.png)
## 设计理念描述：
**核心图形：** Logo 整体呈现一个向上的盾牌形状，象征着软件工程的稳固和团队的向上发展愿景。
**元素融合：** 盾牌由多个相互交织的几何色块构成，色块上分别融入了代表“前端/UI”的网页线框、代表“后端/DB”的数据库圆柱体、以及代表“人工智能/AIGC”的神经网络节点和微小发光的 AI 核心。
**连接与合作：** 三条相互连接的向上 Git 分支线条将这些技术元素绑定在一起，象征着团队成员紧密合作、版本控制和项目的持续推进。
**色彩与风格：** 采用了科技蓝、翠绿、深蓝和金黄的现代配色，呈现出智能、专业、合作和创新的氛围。
**文字清晰：** 图形下方使用清晰的现代中英文书写了 软件工程团队 SE'24 以及全称 广东工业大学计算机学院，确保了识别度。
