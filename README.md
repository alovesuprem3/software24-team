# 团队介绍
**欢迎来到我们的项目专属基地！我们是一支充满激情、致力于将奇思妙想转化为现实代码的小分队。**

在这里，我们坚信：
  + 我们对待每一个 Warning 都像对待 Error 一样严谨，绝不让潜伏的 Bug 活过今晚 🗡️
  + 沟通不仅靠群聊，更靠规范的接口文档和默契的进度同步 🤝
  + 拒绝“个人英雄主义”，每个人都是不可或缺的齿轮，咬合驱动项目稳步前行，坚决告别 DDL 前的极限熬夜 ⏱️
  + 我们认为代码不仅是写给机器执行的，更是写给人类阅读的艺术品 🎨

😜 我们相聚于此，目标不仅是高分拿下这门课程，更是为了在实战中淬炼技术，结下深厚的“革命友谊”（顺便比比谁的键盘敲得更响）。

🔗 **我们的信条**

* Talk is cheap. Show me the code. (并且要附带完整的注释)。
* Merge 代码前不 Review，运行报错两行泪。
* 不怕遇到天坑需求，就怕团队没有团魂。
